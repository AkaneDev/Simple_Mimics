# Simple Mimics
This mod adds Mimics to your game. They can copy Player Nametags, Skins, and even voice clips through [Simple Voice Chat](https://modrinth.com/plugin/simple-voice-chat) (Required).

It requires no additional setup and is ideal for use with friends or even playing alone.

---

## Why would I make this?

I was bored and created this just to mess with my friends. After making it, I thought: "Hey, maybe the world would enjoy this too."

## How can I use this mod?

You are allowed to use this mod in:

- Modpacks
- Videos
- Streams
- VODs

### Videos

For videos, please credit me, **AkaneDev**, in any way possible. This can be done through:

- A pinned comment
- A small pop-up in the video
- A link to the mod in the description
- Mentioning the mod name and author in the description

### Streams

For streams, please credit me, **AkaneDev**, in any way possible. This can be done through:

- Mentioning the mod during the stream
- Including the mod name and author in the stream description
- Linking the mod page in the stream description
- A command to show the mod name in chat, or any other chat based credit 

### VODs

VODs are treated differently from live streams and videos. Credit is **not required** for VODs, but it would be greatly appreciated if you could credit **AkaneDev** where possible.

Examples of appreciated credit:

- A link to the mod page in the description
- Mentioning the mod name and author
- Adding it to a list of mods used

### Modpacks

If you include this mod in a modpack, it must be listed in the modpack description with credit to **AkaneDev**.

Thank you for supporting the mod!
